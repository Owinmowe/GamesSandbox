Detailed Description

Data class for Gameplay Configurations in BottleGame. This class is directly used by the BottleGame GameplayScreenPresenter to create a logic version of the BottleGame with the data. This class is only modifiable before build time (Scriptable objects don't work the same way on builds that on editor).
